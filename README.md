# 📘 JPA Basic Study - 김영한 강의 실습

이 저장소는 [인프런 - 자바 ORM 표준 JPA 프로그래밍 - 기본편](https://www.inflearn.com/course/ORM-JPA-Basic) 강의를 따라 학습한 내용을 실습하고 정리한 공간입니다.
